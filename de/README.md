# Leitfaden zur Elternschaft
Informationen und Ratschläge für werdende Väter.
Der erste weltweite kollaborative Leitfaden, der Ressourcen zur Vaterschaft sammelt.

# Motivation
Ich habe das Gefühl, dass es trotz der Schwierigkeit, dieses Thema zu diskutieren und sich darüber zu einigen, nützlich ist, eine gute Menge an Informationen zur Verfügung zu haben, die jederzeit abgerufen werden können.

Verschiedene Meinungen, Ressourcen und Buchempfehlungen von anderen Eltern auf der ganzen Welt zu erhalten, unabhängig von Kultur oder Kontext,
zu den besten Praktiken und Methoden der Elternschaft ist immer positiv.
