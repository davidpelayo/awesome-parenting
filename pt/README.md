# Paternidade Consciente
Informações e conselhos para pais expectantes.
O primeiro guia colaborativo mundial que reúne recursos para a paternidade.

# Motivação
Sinto que, embora seja um tema difícil de discutir e chegar a consenso, é útil ter uma boa quantidade de informação disponível para consultar a qualquer momento.

Obter diferentes opiniões, recursos e livros recomendados por outros pais do mundo, independentemente da cultura ou contexto,
sobre as melhores práticas e metodologias de parentalidade, em termos gerais, é sempre positivo.
